Actividad 9 - Neo4j

Proyecto académico desarrollado en Neo4j utilizando bases de datos orientadas a grafos.

El sistema modela un entorno turístico inteligente mediante nodos y relaciones entre usuarios, lugares, ciudades, eventos, alojamientos e intereses.

Se implementaron consultas Cypher para realizar recomendaciones de lugares, eventos y alojamientos según afinidades e interacciones entre usuarios.

Tecnologías utilizadas:

• Neo4j
• Cypher
• GitHub
• CSV
